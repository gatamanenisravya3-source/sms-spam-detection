# SMS Spam Detection using Machine Learning

Jul 2025 – Aug 2025

End-to-end machine learning application to classify SMS messages as spam or ham using NLP techniques.

## Highlights
- Data preprocessing + TF-IDF vectorization, model training with Logistic Regression
- Evaluated using precision, recall, and F1-score; probability threshold tuning to control false positives
- ~98% accuracy
- Deployed as an interactive Streamlit app for real-time SMS classification

## Tech Stack
Python, Scikit-learn, NLP, Streamlit
